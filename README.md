# Smart Mouse Trap – ESP32 Telegram Notification

## Project Description
This project is a smart mouse trap notification system built using an ESP32 microcontroller.
When a mouse enters the trap, an IR sensor detects movement and the ESP32 sends a Telegram message.

## Hardware Used
- ESP32 Development Board
- IR Obstacle Detection Sensor
- Mouse Trap Box
- WiFi Internet Connection

## Software Used
- Arduino IDE
- UniversalTelegramBot Library
- ArduinoJson Library

## How It Works
1. ESP32 connects to WiFi.
2. IR sensor monitors trap activity.
3. If mouse detected, ESP32 sends Telegram alert.
4. User receives notification in Telegram group.

## Applications
- Pest monitoring
- IoT automation projects
- Smart home alerts
